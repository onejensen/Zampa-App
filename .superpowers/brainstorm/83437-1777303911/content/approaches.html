A — Spotlight + Overlay

Overlay oscuro sobre toda la pantalla con un recorte que ilumina el elemento. Tooltip apuntando al elemento con "Siguiente" y "Omitir". El patrón clásico de coach marks.

Pros

  • Máximo foco en el elemento
  • Estándar de industria
  • Aspecto muy pulido

Contras

  • Más complejo de implementar
  • Bloquea toda la UI

B — Ring pulsante + Bottom Sheet

Sin overlay. El elemento destacado tiene un anillo animado. Un panel inferior describe el paso. La UI sigue visible en el fondo.

Pros

  • Menos intrusivo
  • Implementación más simple
  • Usuario ve contexto real

Contras

  • Menos impacto visual
  • Usuario puede confundirse con otros elementos

C — Spotlight suave + Tooltip flotante

Overlay menos opaco (55%). Spotlight sobre el elemento exacto (tab, botón). Tooltip flotante pequeño sin panel inferior. Más ligero visualmente.

Pros

  • Balance entre A y B
  • Funciona bien en tabs y botones pequeños

Contras

  • Spotlight exacto requiere coordenadas del elemento

Mi recomendación: A para los pasos principales + inspiración de C para los tabs de navegación. Pero elige lo que te guste.
